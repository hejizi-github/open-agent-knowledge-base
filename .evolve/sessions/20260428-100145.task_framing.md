# Task Framing

## Goal
持续学习开源 Agent 项目，沉淀可复用知识库，持续产出面向中文读者的深度万字长文；每篇文章必须图文并茂，图片只输出可直接用于生图的提示词，不直接生成图片。

## Wiki Pages Used
- wiki/index.md — 项目 LLM Wiki 入口，目前为空，本轮后建议更新 Sources 段
- library/methodology/_form.md — 新建，承载 Step 0 意图锚定 + Step A 形态识别

## Expert Lens
本轮未直接调用专家模型，但通过逆向工程 Anthropic Research Engineering 和 Harrison Chase 的公开文章，提取了"分层复杂度""模式命名+边界""反共识作为论点"等顶尖写作特征。这些将作为下一轮 Step B 的方法论素材。

## Default Assumptions
- 文章文风默认对标 Anthropic "Building Effective Agents"（实践架构分析导向）
- 读者为有经验的工程师/技术负责人，非入门科普
- 中文深度技术长文的市场对标为 InfoQ 架构头条级别
- 图片策略严格遵守"只输出提示词，不生图"
- 跨项目对比必须 ≥2 独立来源

## This Session Step
Step A（形态识别）：
- 因为本目标是首次进入 self-evolve 循环，经验库为空，必须先识别输出体裁和顶尖执行者
- 已抓取 2 份国际顶尖样本（Anthropic + LangChain）并保存为 raw source
- 已归纳跨样本共性特征和预期间反共识点
