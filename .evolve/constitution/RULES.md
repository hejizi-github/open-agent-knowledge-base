# Project Rules

## Mission

Build a durable Chinese knowledge base about open-source Agent systems, then
turn it into deep long-form essays with image generation prompts.

## Hard Rules

- Do not publish broad claims about a project without current source evidence.
- Do not use generated images directly. Only write image prompts.
- Do not write shallow listicles. Prefer fewer, deeper essays.
- Do not let a single project define the whole argument; cross-project claims
  need comparison.
- Keep raw source notes separate from synthesized knowledge.

## Preferred Rhythm

1. Capture sources for one or two projects.
2. Write project profiles.
3. Extract patterns and anti-patterns.
4. Draft article outline.
5. Expand one section at a time.
6. Add image prompts after the section argument is clear.
7. Publish only after source and structure review.
