# Image Prompts Index

This directory stores prompts for images that support long-form articles.
Do not generate images here; only write prompts.

## Prompt Types

- Cover image
- Architecture diagram
- Comparative matrix
- Timeline / evolution map
- Failure mode illustration
- Conceptual metaphor

## Quality Standard

Each prompt must specify:

- intended article section;
- visual goal;
- composition;
- concrete objects/text labels;
- style;
- aspect ratio;
- negative prompt / avoid list.
