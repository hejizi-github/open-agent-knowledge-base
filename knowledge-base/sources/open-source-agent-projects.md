## Per-Project Source Capture Checklist

- Repository URL, license, primary language, current docs URL.
- Latest release or current default branch status.
- Core architecture concepts.
- Tool execution model.
- Planning/workflow model.
- Memory/state model.
- Observability/evaluation model.
- Example apps and recommended patterns.
- Known limitations from docs/issues.
