# Knowledge Base Index

This knowledge base studies open-source agent projects as engineering systems.

## Project Profiles

- `projects/`: one profile per open-source agent project.

## Cross-Project Patterns

- `patterns/`: reusable design patterns found across projects.
- `anti-patterns/`: repeated failure modes and misleading abstractions.

## Rubrics

- `rubrics/`: evaluation criteria for agent runtimes, workflow engines,
  tool-use layers, memory systems, planning loops, and observability.

## Sources

- `sources/`: source indexes and research notes before synthesis.

## Article Pipeline

1. Capture sources.
2. Write project profiles.
3. Extract cross-project patterns.
4. Build a thesis and outline.
5. Draft a long-form essay.
6. Add image generation prompts.
7. Publish once claims are source-backed.
